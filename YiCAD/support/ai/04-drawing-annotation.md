# 填充、文字与图片操作

YiCAD 支持图案填充（Hatch）、文字标注（单行/多行）和图片插入功能，用于完善图纸的视觉表达和注释。

## 图案填充 (ActionDrawHatch)

在闭合区域内填充图案（如斜线、网格、实体色等）。

- **命令行**：`tc`
- **操作步骤**：
  1. 输入命令，弹出填充设置对话框
  2. 选择填充图案（ANSI、ISO 等标准图案库）
  3. 设置填充比例和角度
  4. 选择填充方式：
     - **拾取点**：在闭合区域内点击一点，系统自动识别边界
     - **选择对象**：手动选择要填充的边界对象
  5. 预览填充效果
  6. 确认后完成填充

- **注意事项**：
  - 填充区域必须是完全闭合的，否则填充会"泄漏"
  - 填充图案存储在 `support/patterns/` 目录的 `.pat` 文件中
  - 支持关联填充：修改边界时填充自动更新

## 文字操作

### 单行文字 (ActionDrawText)

在图纸上放置单行文字。

- **命令行**：`text`、`txt`、`d`、`dhwz`
- **操作步骤**：
  1. 指定文字的插入点
  2. 输入文字高度（或使用默认值）
  3. 输入文字旋转角度（或使用默认值 0）
  4. 在对话框中输入文字内容
  5. 确认后文字放置在图纸上

- **支持的字体格式**：
  - SHX 矢量字体（`support/fonts/` 目录）
  - 系统 TrueType 字体
  - 中文字体：`CHINA.SHX`、`HZFS.SHX`、`hztxt.shx`、`gbcbig.shx`

### 多行文字 / MTEXT (ActionDrawMText)

在图纸上放置支持多行、多格式的文本框。

- **命令行**：`mtext`、`mtxt`、`dhwz`
- **操作步骤**：
  1. 指定文本框的第一个角点
  2. 指定文本框的对角点（定义文字宽度）
  3. 在多行文字编辑器中输入内容
  4. 可使用基本格式（字体、大小、加粗、斜体、颜色）
  5. 确认后文字放置在图纸上

- **与单行文字的区别**：MTEXT 支持自动换行、段落格式，适合较长的文字说明，而单行文字每行是独立实体。

### 分解文字 (ActionModifyExplodeText)

将文字实体分解为基本几何图形（直线段和曲线），便于进一步编辑。

- **命令行**：`fjwz`
- **操作步骤**：
  1. 选择要分解的文字实体
  2. 文字被分解为独立的线段/曲线

## 插入图片 (ActionDrawImage)

在图纸中插入光栅图片（PNG、JPG、BMP 等格式）。

- **命令行**：`tp`
- **操作步骤**：
  1. 输入命令，弹出文件选择对话框
  2. 选择要插入的图片文件
  3. 指定图片的插入点（左下角）
  4. 指定图片的缩放比例或对角点
  5. 图片作为 Image 实体插入图纸

- **注意事项**：
  - 图片以引用方式嵌入（保存在 OCD 文件中）
  - 支持常见的图片格式（PNG、JPG、BMP、TIFF 等）
  - 插入后可进行移动、缩放、旋转等基本操作
