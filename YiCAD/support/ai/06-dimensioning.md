# 尺寸标注操作

YiCAD 支持多种尺寸标注类型，包括线性标注、对齐标注、半径标注、直径标注、角度标注和引线标注。标注自动与实体关联，实体修改时标注值自动更新。

## 线性标注 (ActionDimLinear)

创建水平方向或竖直方向的线性尺寸标注。

- **命令行**：`dimlinear`、`dl`、`dr`、`xxbz`、`xx`
- **操作步骤**：
  1. 指定第一条尺寸界线的起点
  2. 指定第二条尺寸界线的起点
  3. 指定尺寸线的位置（决定标注是水平还是竖直方向）
  4. 标注值自动显示

- **子选项**：
  - 水平方向：尺寸线水平放置
  - 竖直方向：尺寸线竖直放置
  - 旋转：尺寸线按指定角度倾斜

## 对齐标注 (ActionDimAligned)

创建与标注点连线方向对齐的尺寸标注，适用于倾斜的实体。

- **命令行**：`dimaligned`、`da`、`dqbz`、`dq`
- **操作步骤**：
  1. 指定第一条尺寸界线的起点
  2. 指定第二条尺寸界线的起点
  3. 指定尺寸线的位置
  4. 尺寸线与被标注对象的倾斜方向平行

- **与线性标注的区别**：线性标注总是水平或竖直，而对齐标注的尺寸线与被测对象平行。

## 半径标注 (ActionDimRadial)

标注圆或圆弧的半径。

- **命令行**：`dimradial`、`dimradius`、`bjbz`、`bj`
- **操作步骤**：
  1. 选择一个圆或圆弧
  2. 指定标注文字的位置
  3. 半径标注自动生成，标注文字前带有 R 符号

## 直径标注 (ActionDimDiametric)

标注圆或圆弧的直径。

- **命令行**：`dimdiametric`、`dimdiameter`、`dd`、`zjbz`、`zj`
- **操作步骤**：
  1. 选择一个圆或圆弧
  2. 指定标注文字的位置
  3. 直径标注自动生成，标注文字前带有直径符号

## 角度标注 (ActionDimAngular)

标注两条直线之间的夹角或圆弧的圆心角。

- **命令行**：`dimangular`、`dan`、`jdbz`、`jd`
- **操作步骤**：
  1. 选择第一条直线（或圆弧）
  2. 选择第二条直线（若选择圆弧则自动取其圆心角）
  3. 指定标注弧线的位置
  4. 角度值自动显示

## 引线标注 (ActionDimLeader)

创建带有引线和文字的标注，用于对图纸元素添加注释。

- **命令行**：`dimleader`、`ld`、`yxbz`、`yx`
- **操作步骤**：
  1. 指定引线的起点（箭头指向的位置）
  2. 指定引线的第二点（转折点）
  3. 指定引线的终点（文字位置）
  4. 输入标注文字
  5. 文字沿引线末端放置

## 基线标注 (ActionDimBaseline)

创建一系列从同一基线出发的线性标注。

- **操作步骤**：
  1. 首先创建一个线性标注作为基准
  2. 输入基线标注命令
  3. 依次指定下一个尺寸界线的起点
  4. 每个标注从同一基线开始测量

## 标注样式管理 (ActionDimStyle)

管理标注的外观样式，包括文字大小、箭头样式、尺寸线偏移、单位格式等。

- **操作步骤**：
  1. 打开标注样式管理器
  2. 可创建新样式、修改已有样式
  3. 设置参数：
     - 文字高度和样式
     - 箭头类型和大小
     - 尺寸线颜色和线宽
     - 单位精度和格式
     - 公差显示
  4. 应用样式后，该样式的标注统一更新

## 文字样式管理 (ActionTextStyle)

管理图纸中的文字样式，包括字体、大小、宽高比等。

- **操作步骤**：
  1. 打开文字样式管理器
  2. 创建新样式或修改已有样式
  3. 选择 SHX 字体或系统字体
  4. 设置文字高度、宽度因子、倾斜角度
  5. 应用后，使用该样式的文字统一更新
