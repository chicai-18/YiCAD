# 图块操作

图块（Block）是多个实体的组合，作为一个整体进行插入和管理。图块支持嵌套（块中套块）和属性定义，是提高绘图效率和标准化的重要工具。

## 图块基本概念

- **图块定义（Block Definition）**：图块的模板，存储在图纸的块表中
- **图块引用（Block Reference/Insert）**：图块定义在图纸中的实例，包含位置、缩放、旋转等属性
- **属性（Attribute）**：图块中可变的文字信息，在插入时可填写（如零件编号、名称等）
- **嵌套块**：一个图块定义中可以包含其他图块引用

## 图块创建与管理

### 创建图块 (ActionBlocksCreate)

将选中实体转换为图块定义。

- **操作步骤**：
  1. 选择要组成图块的实体
  2. 输入命令
  3. 指定图块的名称
  4. 指定图块的基点（插入时的参考点，通常选择图形的中心或角点）
  5. 选中实体被替换为图块引用，原实体成为图块定义的一部分

- **注意事项**：
  - 基点位置很重要，影响后续插入的位置精度
  - 图块名称在同一图纸内必须唯一

### 保存图块 (ActionBlocksSave)

将图块定义保存到外部文件，供其他图纸使用。

- **操作步骤**：
  1. 选择要保存的图块（从图块列表中选择）
  2. 指定保存路径和文件名
  3. 图块以 OCD 格式保存为独立文件

### 另存为 (ActionBlocksSaveAs)

将当前图块定义以不同名称另存。

### 插入图块 (ActionBlocksInsert)

将已有的图块定义插入到图纸中。

- **操作步骤**：
  1. 从图块列表中选择要插入的图块
  2. 指定插入点
  3. 指定 X 方向缩放比例（默认 1.0）
  4. 指定 Y 方向缩放比例（默认等于 X）
  5. 指定旋转角度（默认 0）
  6. 如果图块包含属性，弹出属性填写对话框

### 插入准备 (ActionBlockInsertPrepare)

选择图块后进入悬浮预览状态，在图纸中动态预览插入效果。

### 删除图块 (ActionBlocksDelete)

从块表中删除图块定义（需先删除所有该图块的引用）。

### 编辑图块 (ActionBlocksEdit)

进入图块编辑模式，在图块上下文中修改图块定义的内容。

- **操作步骤**：
  1. 双击图块引用或选择编辑命令
  2. 进入块编辑环境（其他实体变灰）
  3. 修改图块内的实体
  4. 保存并退出块编辑器
  5. 该图块的所有引用自动更新

### 导入图块 (ActionBlocksImport)

从外部 OCD 文件导入图块定义到当前图纸。

### 定义属性 (ActionDefineAttributes)

为图块定义可编辑的文字属性。

- **操作步骤**：
  1. 在块编辑环境中
  2. 定义属性：名称、提示文字、默认值
  3. 指定属性的插入位置
  4. 保存图块后，插入该图块时会提示填写属性值

## 图块分解 (ActionBlocksExplode)

将图块引用分解为组成实体。

- **命令行**：`explode`、`xp`、`fj`
- **操作步骤**：
  1. 选择图块引用
  2. 图块被分解为原始组成实体
  3. 嵌套图块需要多次分解才能完全展开
