# 图形修改操作

YiCAD 提供全面的图形修改工具，包括移动、复制、旋转、缩放、镜像、修剪、延伸、偏移、倒角、圆角、打断、拉伸、分解和反向等操作。

## 基本变换

### 移动 (ActionModifyMove)

将选中的实体从一个位置移动到另一个位置。

- **命令行**：`move`、`mv`、`fz`
- **操作步骤**：
  1. 选择要移动的实体（可多选）
  2. 按 Enter 确认选择
  3. 指定移动的基点
  4. 指定移动的目标点

### 复制 (ActionModifyCopy)

复制选中的实体到新位置，原实体保留。

- **命令行**：`copy`、`co`、`cp`
- **操作步骤**：
  1. 选择要复制的实体
  2. 按 Enter 确认选择
  3. 指定复制的基点
  4. 指定复制的目标点
  5. 可继续指定多个目标点进行多次复制
  6. 按 Esc 或右键结束

### 旋转 (ActionModifyRotate)

围绕指定基点旋转选中的实体。

- **命令行**：`rotate`、`ro`、`xz`
- **操作步骤**：
  1. 选择要旋转的实体
  2. 指定旋转基点（旋转中心）
  3. 输入旋转角度（正值逆时针，负值顺时针）
  4. 或通过鼠标拖拽动态确定角度

### 缩放 (ActionModifyScale)

以指定基点为中心缩放选中的实体。

- **命令行**：`scale`、`sz`、`sf`
- **操作步骤**：
  1. 选择要缩放的实体
  2. 指定缩放基点
  3. 输入缩放比例因子（>1 放大，<1 缩小）
  4. 或通过鼠标拖拽动态确定比例

### 镜像 (ActionModifyMirror)

以指定直线为对称轴，创建选中实体的镜像副本。

- **命令行**：`mirror`、`mi`、`jx`
- **操作步骤**：
  1. 选择要镜像的实体
  2. 指定镜像轴的第一点
  3. 指定镜像轴的第二点
  4. 选择是否删除原对象（默认保留）

### 移动并旋转 (ActionModifyMoveRotate)

一次操作中同时完成移动和旋转。

- **操作步骤**：
  1. 选择实体
  2. 指定移动基点和目标点
  3. 指定旋转角度

## 修剪与延伸

### 修剪 (ActionModifyTrim)

以剪切边为界，修剪掉实体超出边界的部分。

- **命令行**：`trim`、`tm`、`xz`
- **操作步骤**：
  1. 选择剪切边（作为修剪边界的实体）
  2. 按 Enter 确认
  3. 点击要修剪掉的实体部分（点击哪边就修剪哪边）
  4. 可连续修剪多个位置
  5. 按 Esc 或右键结束

### 延伸 (ActionModifyExtend)

将实体延伸到指定的边界。

- **命令行**：`yc`
- **操作步骤**：
  1. 选择边界实体（延伸的目标边界）
  2. 按 Enter 确认
  3. 点击要延伸的实体的端部（点击哪端就延伸哪端）
  4. 实体自动延伸到最近的边界

### 偏移 (ActionModifySingleOffset)

创建实体的平行副本，偏移指定距离。

- **命令行**：`py`
- **操作步骤**：
  1. 输入偏移距离
  2. 选择要偏移的实体（直线、圆、弧、多段线）
  3. 在实体一侧点击，确定偏移方向
  4. 偏移副本创建完成

## 倒角与圆角

### 倒角 (ActionModifyBevel)

在两条直线的交点处创建斜角（倒角）。

- **命令行**：`bevel`、`bev`、`ch`、`dj`
- **操作步骤**：
  1. 设置倒角参数：
     - 距离方式：指定两个倒角距离
     - 角度方式：指定倒角距离和角度
  2. 选择第一条直线
  3. 选择第二条直线
  4. 倒角在两条直线之间创建

### 圆角 (ActionModifyRound)

在两条直线（或曲线）的交点处创建圆角。

- **命令行**：`fillet`、`fi`、`yj`
- **操作步骤**：
  1. 设置圆角半径
  2. 选择第一个对象
  3. 选择第二个对象
  4. 自动创建圆角弧

## 打断与拉伸

### 打断 (ActionModifyCut)

在指定点将实体打断为两部分。

- **命令行**：`divide`、`cut`、`div`、`di`、`dd`
- **操作步骤**：
  1. 选择要打断的实体
  2. 指定打断点位置
  3. 实体在打断点处被分成两个独立实体

### 两点打断 (ActionModifyCut2P)

在两点之间打断实体，删除两点之间的部分。

- **命令行**：`lddd`
- **操作步骤**：
  1. 选择要打断的实体
  2. 指定第一个打断点
  3. 指定第二个打断点
  4. 两点之间的部分被删除，或实体在两点处断开

### 拉伸 (ActionModifyStretch)

通过移动选中顶点来拉伸实体。

- **命令行**：`stretch`、`ss`、`ls`
- **操作步骤**：
  1. 使用交叉窗口选择要拉伸的顶点
  2. 按 Enter 确认
  3. 指定拉伸基点
  4. 指定拉伸目标点
  5. 选中范围内的顶点移动，范围外的顶点保持不动

## 其他修改操作

### 删除

删除选中的实体。可通过选择实体后按 Delete 键触发。

### 分解 (ActionBlocksExplode)

将复合实体（图块、多段线、填充、标注等）分解为基本组成实体。

- **命令行**：`explode`、`xp`、`fj`
- **操作步骤**：
  1. 选择要分解的复合实体
  2. 实体被分解为基本元素（如多段线分解为直线段和弧段）

### 反向 (ActionModifyReverse)

反转实体的方向（如直线的起点和终点互换，多段线的顶点顺序反转）。

- **命令行**：`reverse`、`fx`

### 反转 (ActionModifyRevertDirection)

回退实体的方向修改。

- **命令行**：`revert`、`rev`、`fz`

### 特性编辑 (ActionModifyEntity)

查看和修改选中实体的属性（图层、颜色、线型、线宽等）。

- **命令行**：`properties`、`prop`、`mp`、`tx`
- **操作步骤**：
  1. 选择要编辑的实体
  2. 弹出特性面板，显示实体的所有属性
  3. 修改需要的属性值
  4. 修改即时生效

### 矩形阵列 (ActionArrayRect)

将选中实体按矩形网格方式阵列复制。

- **命令行**：`arrayrect`、`ar`、`jz`
- **操作步骤**：
  1. 选择要阵列的实体
  2. 设置行数和列数
  3. 设置行间距和列间距
  4. 确认后生成矩形阵列
