# 捕捉与限制操作

捕捉（Snap）是 CAD 绘图的核心辅助功能，用于精确地将光标锁定到已有实体的特定几何点上。限制（Restrict）用于约束光标移动方向，确保绘制精度。

## 捕捉模式

### 自由捕捉 (ActionSnapFree)

不启用任何捕捉，光标在画布上完全自由移动。

- 适用于不需要精确对齐的场景
- 可以通过快捷键随时切换

### 网格捕捉 (ActionSnapGrid)

光标吸附到最近的网格交点上。

- 网格间距可在图纸设置中调整
- 适用于规则布局的快速绘图

### 端点捕捉 (ActionSnapEndpoint)

光标自动吸附到直线、多段线、圆弧等实体的端点上。

- 当光标靠近端点时，会显示捕捉标记（小方框）
- 这是最常用的捕捉方式之一

### 线上捕捉 (ActionSnapOnEntity)

光标自动吸附到实体上的最近点。

- 当光标靠近任何实体时，捕捉到实体的最近点
- 适用于需要在实体上但不一定是特定点的位置

### 圆心捕捉 (ActionSnapCenter)

光标自动吸附到圆或圆弧的圆心。

- 当光标靠近圆周或弧线时，自动定位到圆心
- 显示圆心十字标记

### 中点捕捉 (ActionSnapMiddle)

光标自动吸附到直线段或弧段的中点。

- 当光标靠近实体的中点位置时触发
- 显示三角形中点标记

### 交点捕捉 (ActionSnapIntersection)

光标自动吸附到两个实体的交点上。

- 同时检测两条实体相交的位置
- 即使实体不直接相交，也检测其延长线的交点

## 方向限制

### 无限制 (ActionRestrictNothing)

不限制光标的移动方向。

### 正交限制 (ActionRestrictOrthogonal)

将光标移动限制在水平或竖直方向上（相对于上一个输入点）。

- **常用快捷键**：F8
- 绘制水平线或竖直线时非常有用
- 状态栏有正交模式开关显示

### 水平限制 (ActionRestrictHorizontal)

将光标移动限制在纯水平方向。

### 竖直限制 (ActionRestrictVertical)

将光标移动限制在纯竖直方向。

## 状态栏控制

YiCAD 底部状态栏提供以下功能的快速开关：

| 开关 | 功能 |
|------|------|
| **网格 (Grid)** | 显示/隐藏背景网格 |
| **捕捉 (Snap)** | 开启/关闭捕捉功能 |
| **正交 (Ortho)** | 开启/关闭正交限制模式 |
| **显示线宽** | 开启/关闭线宽的屏幕显示 |
| **单位显示** | 切换坐标显示的单位格式 |
