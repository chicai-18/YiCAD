# YiCAD 软件概述

YiCAD 是一款开源的二维 CAD（计算机辅助设计）软件，基于 C++23 和 Qt 5.15 开发，使用 OpenGL 进行高性能图形渲染。软件采用类似 AutoCAD 的 Ribbon 界面风格，支持 OCD 原生格式（基于 XML 的压缩格式），兼容 AutoCAD 的线型（.lin）、填充图案（.pat）和字体（.shx）文件。

## 系统架构

YiCAD 采用 MVC + Action 架构模式：

- **数据模型（Model）**：`kernel/data_model/` 中的实体数据类，包括直线、圆、弧、椭圆、多段线、样条线、填充、图片、标注等所有 CAD 实体类型。
- **视图（View）**：`GuiDocumentView`（基于 `QOpenGLWidget`）提供四层渲染——背景层（网格）、文档层（实体）、预览层（交互预览）和前景层（光标和捕捉指示）。
- **控制（Controller/Action）**：约 125 个 `ActionInterface` 子类，每个对应一个用户操作（画线、移动、标注等）。
- **命令系统**：`Commands` 单例将命令字符串映射到具体 Action。
- **撤销/重做**：基于命令栈的事务机制，支持宏命令和实体表跟踪。

## 主要功能领域

| 领域 | 说明 |
|------|------|
| **绘图** | 直线、圆、弧、椭圆、多段线、样条线、点、射线、构造线、云线、填充、图片 |
| **修改** | 移动、复制、旋转、缩放、镜像、修剪、延伸、偏移、倒角、圆角、打断、拉伸、分解、反向 |
| **标注** | 线性、对齐、半径、直径、角度、引线、基线标注，标注样式管理 |
| **文字** | 单行文字、多行文字（MTEXT），文字样式管理 |
| **图层** | 新建、删除、重命名、激活、冻结/解冻、锁定/解锁、图层颜色/线型/线宽 |
| **图块** | 创建、插入、编辑、保存、导入、删除图块，属性定义 |
| **捕捉** | 自由、网格、端点、线上、圆心、中点、交点捕捉，正交/水平/竖直限制 |
| **测量** | 点对点距离、实体到点距离、两线夹角、选中实体总长、多边形面积 |
| **视图** | 缩放、平移、网格显示、草稿模式 |
| **文件** | 新建、打开、保存、另存为、导出图片、打印、PDF 导出 |
| **AI 助手** | 基于 DeepSeek 大语言模型的自然语言建模、RAG 知识库问答 |

## 界面布局

YiCAD 使用 Ribbon 风格界面，顶部为功能区（Ribbon），包含"文件"、"Draw2d"、"Solver"、"Options"等分类标签页，以及根据上下文显示的标签页。底部为命令行输入控件和状态栏。中央为 MDI 多文档区域，支持同时打开多个图纸。

命令行支持输入命令别名（如 `line`、`l`、`zx`）来调用功能，也支持拼音简写。

## 文件格式支持

- **OCD（原生格式）**：基于 XML 的压缩格式（gzip），是默认保存格式。
- **JSON 导出**：用于数据交换。
- **图片导出**：支持将图纸导出为常见图片格式。
- **DXF/DWG**：当前版本暂不直接支持，可通过插件扩展。

## 系统要求

- Windows 10/11（主要平台）
- Linux（计划支持）
- OpenGL 3.3+ 显卡
