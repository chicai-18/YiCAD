# AI 助手功能

YiCAD 集成了基于大语言模型（LLM）的 AI 助手，支持自然语言交互进行 CAD 建模和知识问答。目前使用 DeepSeek 作为后端 LLM 服务。

## 功能概述

AI 助手提供两种主要交互模式：

1. **建模模式（Modeling）**：用户用自然语言描述想要绘制的图形，AI 自动解析意图并执行相应的 CAD 操作。
2. **问答模式（QA）**：用户询问软件操作方法，AI 从知识库中检索相关信息并给出操作指导。

## AI 对话框 (AIDialog)

通过 Ribbon 界面或快捷键打开 AI 对话框，在对话框中输入问题或指令。

- **建模模式示例**：
  - "画一个半径为 50 的圆"
  - "在 (100, 200) 处画一个矩形，宽 300 高 200"
  - "把选中的圆向右移动 100 个单位"
  - "删除图层 1 上的所有实体"

- **问答模式示例**：
  - "修剪命令怎么用？"
  - "如何创建图块？"
  - "偏移和镜像有什么区别？"
  - "怎么设置标注样式？"

## AI 架构

### 意图路由器 (AIIntentRouter)

分析用户输入，判断用户的意图类型（问答还是建模操作），将请求分发到对应的处理管线。

- 支持中英文混合输入
- 关键词权重匹配：建模类关键词（画、绘制、创建、移动等）和问答类关键词（怎么、如何、是什么、为什么等）
- 实体名词识别：自动识别用户提到的 CAD 实体类型

### RAG 管线 (RAGPipeline)

检索增强生成（RAG）管线用于问答模式，从软件操作知识库中检索相关内容。

- **知识源**：`support/ai/` 目录中的操作知识库文档
- **检索方式**：基于关键词的检索（后续可升级为向量检索）
- **分块策略**：按 Markdown 标题和段落切分，每块 200-500 字符
- **引用机制**：回答中附带知识来源引用

### LLM 命令桥 (LLMCommandBridge)

将 LLM 的自然语言输出解析为结构化的 CAD 操作命令（JSON 协议）。

- 支持的 JSON 命令类型：
  - 绘制命令：直线、圆、弧、矩形、椭圆、多段线等
  - 修改命令：移动、复制、旋转、缩放等
  - 查询命令：距离、面积等

### 直接实体执行器 (DirectEntityExecutor)

将 LLM 命令桥解析出的 JSON 命令直接转换为 CAD 实体操作，调用对应的 Action 执行。

### 修改执行器 (ModificationExecutor)

处理修改类操作，支持对已有实体的选择和变换。

### 上下文解析器 (ContextResolver)

维护对话上下文，支持连续多轮对话中的引用消解。例如：
- 用户："画一个圆" → AI 创建圆
- 用户："把它移到右边" → AI 理解"它"指的是刚才画的圆

## 配置

### LLM 设置 (LLMSettingsService)

- API 密钥配置（加密存储）
- 模型选择
- API 端点地址
- 温度参数等模型设置

### 密钥存储 (SecretStorage)

API 密钥使用加密方式存储，确保安全性。
