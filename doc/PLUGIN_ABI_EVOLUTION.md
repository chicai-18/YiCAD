# YiCAD 插件 ABI 维护规则

当前 SDK 尚未对外发布，只支持 `YICAD_PLUGIN_ABI_V3`。宿主与插件必须同时报告 v3，
不执行旧版本协商、函数表裁剪或能力降级。

阶段 2 需要的导入和只读接口直接纳入 v3：`YiCadImportApi` 提供原子导入会话，
`YiCadReadApi` 提供文档设置、资源、块和一等实体枚举。后续在首次稳定发布前修改
接口时，应同步更新 ABI 布局断言、宿主实现、C++ SDK 和参考文档。

插件入口、调用约定、固定宽度类型、UTF-8 字符串规则和宿主持有句柄的所有权规则
不得由插件自行改变。异常不得穿过 DLL 边界。
